# Contributing · 贡献指南

Thanks for your interest! 感谢你的关注！

## Asking questions · 提问
- General questions → open a **Discussion**. · 一般性问题请开 **Discussion**。
- Found a bug → open a **Bug report** issue. · 发现问题请开 **Bug report**。
- Have an idea → open a **Feature request** issue. · 有想法请开 **Feature request**。

Please search existing issues first to avoid duplicates. 提交前请先搜索是否已有相同问题。

## Running locally · 本地运行
This is a single static `index.html` — no build step. 这是一个纯静态页面，无需构建。
1. Open `index.html` over **https** (e.g. GitHub Pages / a local server). PWA & live data need https. · 用 **https** 打开（PWA 和实时数据需要 https）。
2. Get a free key at [finnhub.io/register](https://finnhub.io/register) and paste it in the bottom bar. · 在 [finnhub.io](https://finnhub.io/register) 拿免费 Key 填入底部。

> ⚠️ Never commit your API key. It lives only in the browser's localStorage. · 切勿把 API Key 提交到仓库，它只存在浏览器本地。

## Pull requests · 提交代码
1. Fork & create a branch. · Fork 并新建分支。
2. Keep changes focused and described clearly. · 改动聚焦、描述清楚。
3. Test in both Chinese and English, light and dark mode. · 中英文 + 明暗两种模式都测一下。

## Code of conduct · 行为准则
Be kind and constructive. 友善、有建设性。
